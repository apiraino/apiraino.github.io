Rust is a systems programming language focused on three goals:

  • Safety: achieved thanks to a painfully nick-picking compiler
  • Speed: compiled. No gc, no VM, no bytecode
  • Concurrency: it's a bit complicated ...

Updated: April 11, 2018 1:19:22 PM
Type: Potentially Unwanted App
Infection Length: Varies
Risk Impact: Low
Systems Affected: Windows

SUMMARY

Behavior
Wasm.Webcoinminer is a detection for a module of a JavaScript cryptocurrency
miner that runs in web browsers. 


TECHNICAL DESCRIPTION

If WebAssembly (WASM) is supported by the user's browser, this risk is loaded
as a module of a JavaScript coinminer that runs in web browsers. It provides
APIs for JS.Webcoinminer to take full advantage of the victim's hardware 
capability in-browser. 

The JavaScript coinminer is loaded in the browser when the user visits a web
page hosting the JavaScript. It runs as long as the user stays on the web page.
